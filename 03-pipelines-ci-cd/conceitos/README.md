# 📘 Conceitos — Módulo 03: Pipelines CI/CD. 

Este diretório reúne os **conceitos fundamentais de CI/CD**, apresentados de forma progressiva e conectados à prática.

O objetivo aqui não é apenas explicar termos, mas construir o **modelo mental necessário para entender pipelines como parte do fluxo de entrega de software**.

---

## 🧭 Organização dos Conteúdos.  

Os conceitos seguem uma sequência lógica:

1. **O que é CI/CD**
   Fundamentos, motivação e problemas que CI/CD resolve.

2. **Pipeline como código**
   Versionamento, rastreabilidade e automação declarativa.

3. **GitHub Actions**
   Ferramenta, contexto e papel dentro do ecossistema GitHub.

4. **Workflows e Jobs**
   Estrutura de pipelines, eventos, jobs e steps.

Cada arquivo aprofunda um desses pontos, sempre conectando teoria, prática e impacto em ambientes DevOps.

---

## 🧠 Relação com os Módulos Anteriores

* **Módulo 01**: scripts e automação local
* **Módulo 02**: Git como fonte da verdade
* **Módulo 03**: automação contínua e reprodutível

CI/CD é a evolução natural da automação iniciada localmente.
